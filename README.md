NearAlert is a lightweight Vehicle-to-Everything (V2X) collision avoidance 
system designed for low-cost, infrastructure-free deployment on rickshaws, 
CNGs, and motorcycles in dense urban environments like Dhaka, Bangladesh.

This repo contains:
- A full academic proposal (LaTeX + compiled PDF) submitted for CSE 4511: Computer Networks
- An interactive browser-based simulation of the NearAlert VANET protocol,
  featuring real-time animated vehicle movement, UDP beacon modelling,
  Time-to-Closest-Approach (TCA) collision prediction, and live telemetry charts

The system is inspired by aviation's TCAS but re-engineered for ~$13/node 
hardware (ESP32 + NEO-6M GPS), zero cellular dependency, and untrained operators.
